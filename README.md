# Study Planner

Asimple and interactive Study Planner web application built using **HTML**, **CSS**, and **JavaScript**. It helps students organize their study tasks, track their progress, and save tasks using the browser's Local Storage.

---

## Features

- Add new study tasks
- Delete tasks
- Mark tasks as complete
- Save tasks using Local Storage
- Automatically load saved tasks after page refresh
- Display total, completed, and pending tasks
- Progress bar to track study progress
- Search tasks
- Filter tasks by priority
- Filter tasks by completion status
- Color-coded task priorities

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
